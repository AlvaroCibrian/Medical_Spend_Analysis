2. Cómo funciona K-means (idea + fórmula)

K-means divide los datos en K grupos. Cada grupo tiene un centroide (su punto central). El algoritmo repite dos pasos hasta estabilizarse (Lloyd, 1982):

  1. Asignación: cada punto se une al centroide más cercano (distancia euclidiana).
  2. Actualización: cada centroide se recalcula como el promedio de sus puntos.

El objetivo que minimiza es la inercia o suma de cuadrados intra-cluster (WCSS):

INERCIA / WCSS (lo que K-means minimiza)
WCSS = Σk=1..K Σxᵢ ∈ Cₖ ‖ xᵢ − μₖ ‖²

En español: para cada cluster Cₖ con centroide μₖ, se suman las distancias al cuadrado de sus puntos al centro; luego se suma sobre todos los clusters. Cuanto menor es la inercia, más «apretados» están los grupos. La distancia euclidiana entre dos puntos es √Σ(aᵢ−bᵢ)².

3. Paso obligatorio: escalar los datos (StandardScaler)

¡No te saltes esto! K-means usa distancias euclidianas. Si no se escala, charges (que llega a 64 000) dominaría por completo a age (máx 64) y bmi (máx 53): el clustering agruparía prácticamente solo por gasto. Escalar pone a todas las variables en igualdad de condiciones (IBM, s.f.; documentación de scikit-learn).

ESTANDARIZACIÓN Z-SCORE (StandardScaler)
z = ( x − μ ) / σ

En español: a cada valor se le resta la media (μ) de su columna y se divide entre su desviación estándar (σ). El resultado: cada variable queda con media 0 y desviación 1, sin unidades, comparables entre sí.

4. Elegir K: el método del codo (elbow)

¿Cuántos grupos? El método del codo (Thorndike, 1953) prueba varios valores de K y grafica la inercia. Al aumentar K la inercia siempre baja; se busca el «codo»: el punto donde bajar más deja de aportar mucho.

from sklearn.preprocessing import StandardScaler
from sklearn.cluster import KMeans
import matplotlib.pyplot as plt

X = df[["age", "bmi", "charges"]]
X_scaled = StandardScaler().fit_transform(X)     # media 0, desv 1

inercias = []
for k in range(1, 11):
    km = KMeans(n_clusters=k, random_state=42, n_init=10).fit(X_scaled)
    inercias.append(km.inertia_)
plt.plot(range(1, 11), inercias, marker="o")
plt.xlabel("Número de clusters (K)"); plt.ylabel("Inercia (WCSS)")
plt.title("Método del codo"); plt.savefig("../reports/figures/codo.png", dpi=150, bbox_inches="tight"); plt.show()

Cómo leer la gráfica del codo

Es una curva descendente. Busca dónde el descenso «se dobla» formando un ángulo, como un codo. Ese K es el elegido. Para estas variables suele salir K = 3 o 4.

Valida con la silueta (más fiable). El codo a veces no es claro. El coeficiente de silueta (Rousseeuw, 1987) va de −1 a +1 y mide qué tan bien separado está cada punto de su grupo respecto a los vecinos; se elige el K con mayor silueta promedio. Si codo y silueta discrepan, se prefiere la silueta.

from sklearn.metrics import silhouette_score
for k in range(2, 7):
    lab = KMeans(n_clusters=k, random_state=42, n_init=10).fit_predict(X_scaled)
    print(k, silhouette_score(X_scaled, lab))

5. Aplicar K-means y visualizar

k_opt = 3   # el que indique el codo/silueta
km = KMeans(n_clusters=k_opt, random_state=42, n_init=10)
df["cluster"] = km.fit_predict(X_scaled)     # etiqueta de grupo por persona

import seaborn as sns
sns.scatterplot(x="bmi", y="charges", hue="cluster", palette="Set1", data=df)
plt.title("Clusters de asegurados (IMC vs Cargos)")
plt.savefig("../reports/figures/clusters.png", dpi=150, bbox_inches="tight"); plt.show()

print(df.groupby("cluster")[["age","bmi","charges"]].mean())   # perfil de cada grupo
Detalles del código: random_state=42 hace el resultado reproducible; n_init=10 corre el algoritmo 10 veces con distintos inicios y se queda con el mejor (K-means depende del punto de partida). Se grafica con las variables sin escalar para que los ejes sean interpretables (dólares, IMC), pero el agrupamiento se hizo con los datos escalados.

6. Interpretar los clusters (lo más importante)

Un cluster sin interpretar no vale. Usa groupby("cluster").mean() para leer el perfil promedio de cada grupo y ponle nombre. Ejemplo del tipo de resultado esperado:

ClusterPerfil promedioNombre sugerido
0Jóvenes, IMC normal, cargos bajosBajo riesgo
1Edad media, IMC alto, cargos moderados-altosRiesgo por IMC
2Cargos muy elevados (suele coincidir con fumadores)Alto riesgo / costo elevado

Los perfiles exactos dependen del K y de la corrida; describe los tuyos según lo que muestre tu groupby.

Limitaciones honestas (decláralas). (1) K-means asume grupos «esféricos» de tamaño similar y es sensible a outliers; con charges sesgada, un grupo puede quedar dominado por los gastos extremos. (2) El resultado depende de qué variables elijas y de la escala. (3) Los grupos son estadísticos, no categorías médicas oficiales. Menciónalo en la interpretación.
